# JUDUL PROJECT : Alat Konversi Mata Uang 
# DESKRIPSI SINGKAT : Project yang kami buat yaitu konversi mata uang yang sangat memudahkan penggunanya dalam mengetahui nilai tukar mata uang. Kami menggunakan 5 mata uang negara yang berbeda yaitu Rupiah (IDR), Dollar (USD), Euro (EUR), Yen (JPY), dan Dollar Singapore (SGD).
# CARA MENGGUNAKAN : Pertama tama masukkan mata uang asal terlebih dahulu (mata uang sebelum dikonversi), lalu masukkan mata uang tujuan, setelah semuanya lengkap masukkan jumlah uang yang ingin dikonversikan, lalu klik enter untuk mengkonversi, maka hasilnya akan muncul sesuai dengan mata uang yang dituju.
# TEKNOLOGI YANG DIGUNAKAN : Pemograman python dengan struktur berbasis dictionary untuk menyimpan data nilai tukar mata uang.
